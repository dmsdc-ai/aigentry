# Contributing to aigentry

Thanks for your interest in contributing to aigentry!

## Ecosystem Structure

aigentry is a multi-repo ecosystem. Each project has its own contribution guidelines:

- **[aigentry-brain](https://github.com/aigentry/aigentry-brain)** — Memory MCP server
- **[aigentry-deliberation](https://github.com/aigentry/aigentry-deliberation)** — Multi-AI debate framework
- **[aigentry-registry](https://github.com/aigentry/aigentry-registry)** — Agent evaluation platform

## This Repository

This repo contains ecosystem-level resources:

- Brand assets and design guidelines
- Architecture documentation
- Marketing and content materials

## How to Contribute

1. **Report issues** — Open an issue in the relevant sub-project repo
2. **Propose features** — Start a discussion in the relevant repo
3. **Brand assets** — Submit a PR to this repo's `brand/` directory
4. **Documentation** — PRs welcome for `docs/` improvements

## Code of Conduct

Be respectful. Be constructive. Build things that matter.
