# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in any aigentry project, please report it responsibly.

**Do not open a public issue.**

Email: security@aigentry.dev

We will acknowledge receipt within 48 hours and provide a timeline for a fix.

## Scope

This policy covers all repositories under the aigentry organization:

- aigentry-brain
- aigentry-deliberation
- aigentry-registry
