# aigentry Brand Guidelines

## Mascot

The aigentry mascot is `·⣿·` — two cyan dot eyes with a gold braille core.

- Eyes: `#06b6d4` (cyan-500)
- Core: `#d4a574` (gold)
- The mascot represents an entity that observes and processes.

## Color Palette

| Name | Hex | Usage |
|------|-----|-------|
| Gold | `#d4a574` | Primary brand, mascot core, brain |
| Cyan | `#06b6d4` | Mascot eyes, deliberation accent |
| Purple | `#8b5cf6` | Registry, secondary accent |
| Pink | `#ec4899` | Highlight, tertiary accent |
| Dark | `#0a0a0a` | Background |

## Typography

- Primary: **JetBrains Mono** (monospace)
- Fallback: system monospace

## Logo Usage

- Always use on dark backgrounds
- Maintain clear space equal to the height of the `⣿` character
- Do not alter mascot colors or proportions
